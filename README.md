# Vacuum

Vacuum is the world's fastest and most versatile OpenAPI linter and toolkit, built in Go for validating and linting API specifications at scale. It is 100% compatible with Spectral rulesets and supports OpenAPI 3.0, 3.1, and 3.2.

**Human URL:** [https://quobix.com/vacuum](https://quobix.com/vacuum)

**GitHub:** [https://github.com/daveshanley/vacuum](https://github.com/daveshanley/vacuum)

## Description

Vacuum is an ultra-fast OpenAPI linting tool that processes API specifications at lightning speed. It provides interactive dashboards, HTML reports, Language Server Protocol support for IDE integration, and full compatibility with existing Spectral rulesets. Available as a CLI, Go library, npm package, and Docker image.
